# Getting Started

This section is intentionally minimal. Agent-R1 shares the same runtime environment requirements as `verl`, so the goal is not to duplicate infrastructure documentation, but to help you get to a working Agent-R1 setup as quickly as possible.

## In This Section

- [`Installation Guide`](installation-guide.md): use the official `verl` environment setup and confirm what carries over to Agent-R1.
- [`Quick Start`](quick-start.md): download the processed data release or prepare a small GSM8K dataset, then run the single-step sanity check script.

## Recommended Path

1. Set up the environment by following the `verl` installation guide.
2. Download the processed datasets from [ModelScope](https://www.modelscope.cn/datasets/Melmaphother/Agent-R1-data), or regenerate the minimal GSM8K data locally for a quick sanity check.
3. Run the single-step sanity check to confirm that your model path, dependencies, and training stack are wired correctly.
4. Move to the Agent tutorial once the setup is stable.
